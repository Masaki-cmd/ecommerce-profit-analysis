# ecommerce-profit-analysis
# ECサイト販売データ分析: 収益最大化のためのアクションプラン

## 概要
このプロジェクトは、架空のECサイト販売データ（`ecommerce_sales_data.csv`）を使用し、「収益の最大化」をKGI（Key Goal Indicator）として、データ分析から具体的なアクションプラン策定までを行いました。

## 実行された主なステップと決定事項

### 1. データ準備と初期分析
- **データ読み込み**: 提供データ（3500行、7列）を読み込みました。
- **データクリーニング**: 欠損値がないことを確認し、`Order Date`列を`object`型から`datetime`型に変換しました。これにより、時系列分析が可能になりました。
- **カテゴリ別・地域別の売上と利益分析**: 
    - `Electronics`カテゴリと`West`地域が売上・利益ともに最大貢献者であることが判明しました。
    - `Office`カテゴリと`North`地域の売上・利益が低いことが初期課題として特定されました。
    - 全てのセグメントで利益率が約17%と均一だったため、分析の焦点は「売上高の効率的な増加」または「総利益額の小さいセグメントの成長促進」にシフトしました。

### 2. 月次トレンド分析と深掘り
- **月次売上と利益率のトレンド分析**: 月次売上は変動するものの堅調に推移している一方で、**2024年11月に利益率が一時的に約14.5%まで低下**したことを特定しました。
- **2024年11月の利益率低下要因の特定**: 2024年11月の取引詳細を分析し、**「Headphones」「Camera」「Monitor」「Tablet」「Keyboard」といった高売上製品の利益率が著しく低い（9.90%〜13.51%）こと**が主な原因と判明しました。

### 3. 初期アクションプランの策定
分析結果に基づき、以下の初期アクションプランが提案されました。
- **`Office`カテゴリの強化**: 製品拡充、バンドル販売、中小企業向け営業戦略の強化。
- **`Electronics`/`Accessories`のさらなる成長**: 新製品・技術の導入、顧客ロイヤルティ向上、プロモーション強化。
- **`North`地域の深掘り**: 市場調査、限定プロモーション、営業担当者の強化。
- **2024年11月の低利益率製品への対策**: 価格・販促戦略の見直し、サプライチェーン最適化。
- 各施策における「機会費用と投資対効果（ROI）」の意識の重要性が強調されました。

### 4. 新たな着眼点からの分析（全期間における製品別利益率）
- **製品ごとの利益率詳細分析**: 全期間において製品ごとの利益率を分析した結果、以下の主要な発見がありました。
    - 全体平均利益率は約17.29%でした。
    - `Laptop`が最も高利益率（18.47%）であり、`Camera`、`Headphones`、`Monitor`がそれに続きます。
    - `Tablet`が最も低利益率（16.36%）であり、`Printer`、`Smartwatch`、`Smartphone`も全体平均を下回るか同水準でした。
    - 製品間の利益率の差は16%〜18.5%の範囲で比較的小さいものの、個別の改善余地を特定しました。特に、2024年11月に低利益率だった`Tablet`は、全期間で見ても利益率が最も低いことが再確認されました。

### 5. 製品別分析に基づく追加アクションプラン
- **高利益率製品（例: `Laptop`, `Camera`）の強化**: 成功要因の特定と他製品への横展開、プロモーション最適化。
- **低利益率製品（例: `Tablet`, `Printer`）の改善**: コスト構造の徹底分析、価格戦略の見直し、製品の差別化・価値提案の強化。
- **継続的なモニタリング**: 市場やコストの変化へのアジャイルな戦略調整の重要性を提示。

- **詳細な要因分析**: 2024年11月の低利益率製品や、全期間で低利益率の製品について、具体的な販促活動、価格設定、仕入れコストの変動などの詳細な調査が推奨されています。
- **具体的なROI試算**: 提案されたアクションプランに対する具体的なROI試算は、別途詳細な分析や情報が必要とされています。
